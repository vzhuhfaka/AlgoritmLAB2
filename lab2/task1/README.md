## Задание

В этой задаче вы реализуете три основных способа обхода двоичного дерева «в глубину»: центрированный (in-
order), прямой (pre-order) и обратный (post-order). Очень полезно попрактиковаться в их реализации, чтобы лучше

понять бинарные деревья поиска.

Вам дано корневое двоичное дерево. Выведите центрированный (in-order), прямой (pre-order) и обратный (post-
order) обходы в глубину.

## Input / Output

| Input   | Output    |
|---------|-----------|
| 5       | 1 2 3 4 5 | 
| 4 1 2   | 4 2 1 3 5 |
| 2 3 4   | 1 3 2 5 4 |
| 5 -1 -1 |           |
| 1 -1 -1 |           |
| 3 -1 -1 |           |

- Ограничение по памяти. 512 мб.

## Вопросы к задаче

1. Что такое обход двоичного дерева и какие основные виды обходов существуют?<br>
Посещение каждого узла в определенном порядке<br>
Обход в глубину(pre-order, in-order, post-order) <br>
Обход в ширину<br><br>

2. Можете ли вы кратко описать каждый из них?<br>
Pre-order: Посещаем корень, затем левое поддерево, затем правое.<br>
In-order: Посещаем левое поддерево, затем корень, затем правое.<br>
Post-order: Посещаем левое поддерево, затем правое, затем корень.<br>
Обход в ширину (Level-order): Посещаем узлы уровень за уровнем, начиная с корня.<br><br>
